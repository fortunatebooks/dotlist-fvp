# Security Policy

Dotlist FVP stores task data in the browser with `localStorage`. It does not include a server, database, or default network sync.

Exported backups include task titles and work-session history. Treat export files as personal data and remove private task details before sharing them publicly.

Please report security issues by opening a private security advisory on GitHub when available, or by contacting the repository maintainer. Do not include sensitive personal task data in public issues.
